Comment créer un cours bien structuré

Créer un cours bien structuré améliore la lisibilité et l'expérience utilisateur. Cette documentation vous guide sur les balises à utiliser pour structurer votre contenu, ajouter du code, des formules mathématiques et des titres.

Utilisez les balises suivantes pour structurer votre cours :
  • h1"""Titre principal""" pour les titres principaux.
  • h2"""Sous-titre""" pour les sous-titres.
  • h3"""Section""" pour des sections détaillées.

Pour insérer du code avec coloration syntaxique :
  • python"""code en Python""" pour le Python.
  • html"""code en HTML""" pour le HTML.
  • javascript"""code en JavaScript""" pour le JavaScript.
Exemple : 
python"""
def addition(a, b):
    return a + b
"""

Encadrez vos formules mathématiques avec $$ : 
$$ E = mc^2 $$
Cette balise rendra vos formules avec MathJax.

Utilisez la syntaxe suivante :

color"""[color] Mon texte""" 
 Exemple :
color"""[red]Texte rouge"""

Utilisez la balise :

quote"""Ceci est une citation."""

Utilisez \n pour ajouter des sauts de ligne. Exemple : 
Ce texte est sur une ligne.\nEt celui-ci sur une autre.
